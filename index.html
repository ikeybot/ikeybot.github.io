Admin Commands

    Ban

    Bans a user (requires Ban Members permission).

    $ban @user optional reason

    $ban list


    Delete Role

    Deletes a server role (requires Manage Roles permission).

    $deleterole/$dr/$delrole/$drole role name


    Kick

    Kicks a user (requires Kick Members permission).

    $kick @user optional reason


    Mute

    Mutes a user (requires Mute Members permission).

    $mute/$stfu @user 1h/30m/60s optional reason

    $mute list


    Prefix

    Change the prefix for the server (requires Manage Server permission).

    $prefix new prefix

    $prefixreset (in case of issues)


    Purge

    Bulk delete up to 1000 messages (requires Manage Messages permission).

    $purge/$p/$delete 1-1000


    Role

    Adds or removes a role from a user

    $role @user role name


    Unban

    Unbans a user (requires Ban Members permission).

    $unban userID/username


    Unmute

    Unmutes a user (requires Mute Members permission).

    $unmute @user


    Toggle

    Disables commands per channel, or all commands in a channel (requires Admin permission).

    $toggle/$tc command/alias

    $toggle all

Tag Commands

    Create Tag

    Create a tag up to 1950 characters with 8MB img. Unique tags are global, otherwise server tags are made.

    $createtag/$ctag/$ct tagname tag content goes here (+ attch)


    Delete Tag

    Delete a server tag (requires Admin permission or tag owner) or global tag (requires tag owner).

    $deletetag/$dtag/$dt tagname


    Edit Tag

    Edit an existing tag (requires tag owner).

    $edittag/$etag/$et tagname new tag content goes here (+ new attch)


    Gift Tag

    Gift a tag to a user (requires tag owner).

    $gifttag/$gtag/$gt @user tagname


    Random Tag

    Displays a random tag, either server or global

    $randomtag/$rtag/$rt


    Search Tag

    Search for a tag,

    $searchtag/$listtag/$st/$lt tagname/@user/server


    View Tag

    Display a tag, server if available, otherwise global tag.

    $viewtag/$vtag/$vt/$t
